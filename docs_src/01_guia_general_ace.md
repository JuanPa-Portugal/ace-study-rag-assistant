# Guia General de Estudio - Google Cloud ACE

Objetivo: resumir el alcance de la certificacion Associate Cloud Engineer y organizar los dominios de estudio que usara el agente.

Dominios principales:
- Setting up a cloud solution environment: 20%.
- Planning and implementing a cloud solution: 30%.
- Ensuring successful operation: 30%.
- Configuring access and security: 20%.

Servicios clave: Organization, Folders, Projects, IAM, Billing, Compute Engine, Cloud Run, GKE, Cloud Storage, Cloud SQL, BigQuery, VPC, Firewall Rules, Cloud NAT, Cloud Logging y Cloud Monitoring.

Fuentes:
