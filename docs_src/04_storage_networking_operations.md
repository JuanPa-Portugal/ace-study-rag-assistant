# Storage, Networking y Operacion - Google Cloud ACE

Este documento cubre Cloud Storage, Cloud SQL, BigQuery, Firestore, Spanner, VPC, firewall rules, Cloud NAT, Load Balancing, Logging y Monitoring.

Escenarios clave:
- VM privada con salida a internet: Cloud NAT.
- Archivos no estructurados: Cloud Storage.
- Base relacional administrada: Cloud SQL.
- Analitica SQL serverless: BigQuery.
- Alertas por metricas: Cloud Monitoring.

Fuentes:
- https://docs.cloud.google.com/docs
- https://cloud.google.com/cloud-console
