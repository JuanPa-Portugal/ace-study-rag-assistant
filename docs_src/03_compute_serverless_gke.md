# Compute, Serverless y GKE - Google Cloud ACE

Este documento cubre Compute Engine, Managed Instance Groups, Cloud Run, Cloud Run functions y GKE.

Seleccion rapida:
- Compute Engine: control de VM y sistema operativo.
- MIG: VMs replicadas, autoscaling y autohealing.
- Cloud Run: contenedores serverless y aplicaciones stateless.
- Cloud Run functions: funciones event-driven.
- GKE: Kubernetes administrado.
