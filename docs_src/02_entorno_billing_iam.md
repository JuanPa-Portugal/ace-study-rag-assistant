# Entorno, Billing e IAM - Google Cloud ACE

Este documento cubre jerarquia de recursos, proyectos, billing, APIs, cuotas, Cloud Shell, gcloud CLI e IAM basico.

Puntos clave:
- Projects son la unidad operativa principal.
- Billing debe estar vinculado para consumir recursos pagos.
- APIs deben habilitarse antes de usar muchos servicios.
- IAM se basa en principals, roles y policies.
- Least privilege es una idea central para responder escenarios de seguridad.
